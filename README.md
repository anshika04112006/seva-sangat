# SEVA SANGAT

SEVA SANGAT is a smart platform that connects volunteers, NGOs, orphanages, and donors.

Features:
- Secure Login & Registration
- Email OTP Verification
- NGO / Orphanage / Old Age Home Finder
- AI Volunteer Recommendation
- Event Booking System
- Donation Management
- Certificate Download
- Admin Dashboard

Tech Stack:
- React.js
- Node.js
- Express.js
- MongoDB
